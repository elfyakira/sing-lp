### 1-1. コンバージョン戦略（専門家① ダイレクトレスポンスマーケター）

#### ターゲットの心理フロー

```
[広告接触] SNSで「入社祝い金 最大100万円」の文言に目が留まる
    ↓
[FV 3秒] 「100万円」の数字 + 「転職の不安、まるごと解決」→ 自分ごと化
    ↓
[課題共感] 「あ、これ全部自分のことだ」→ スクロール継続の動機
    ↓
[解決策] 祝い金 + サポート + 優良企業 の3点セット → 「ここなら違うかも」
    ↓
[強み詳細] 具体的な数字と仕組みの説明 → 信頼の構築
    ↓
[社会的証明] 実績数字 + 体験談 → 「自分もできそう」
    ↓
[不安解消] FAQ + 完全無料 → 行動障壁の除去
    ↓
[行動] 「とりあえず登録だけでも」→ CV達成
```

#### PASONA法則の適用

| 要素 | セクション | 具体的な役割 |
|------|-----------|-------------|
| Problem | セクション2 | 転職したいのに踏み出せない経済的・心理的ブレーキ |
| Affinity | セクション2 | 「わかります、その不安」という寄り添い |
| Solution | セクション3 | ツナゲル = 祝い金 + 伴走サポート + 優良企業 |
| Offer | セクション4,7 | 最大100万円、完全無料、入社後ケア |
| Narrowing | セクション7 | 東海エリア特化、ツナゲル経由限定 |
| Action | セクション1,9 | 無料会員登録（30秒で完了） |

#### CTA配置戦略

- **FV直下（セクション1）**: 衝動クリック狙い。広告から直結の熱量を逃さない
- **解決策提示後（セクション3末）**: 「これなら解決できそう」の気持ちをすぐ行動に
- **お客様の声後（セクション6末）**: 社会的証明で背中を押された直後
- **最終CTA（セクション9）**: 全情報を読んだ上での理性的判断で最終行動
- **フローティングCTA（モバイル）**: 常時表示で離脱防止

#### メッセージマッチ設計

**広告文言（想定）**: 「入社祝い金 最大100万円｜あなたの転職をまるごと応援」
**FVキャッチ**: 「入社祝い金 最大100万円」をそのまま最大サイズで配置
→ 広告で見た数字がFVに即座に再出現することで「正しいページに来た」と確認させる

### セクション2: 課題提起

#### コピー

**見出し**:
```
こんなことで、転職をあきらめていませんか？
```

**悩みリスト（5項目）**:

1. **「転職したいけど、お金が不安…」**
   引っ越し費用、スーツ代、面接の交通費。転職にはお金がかかる。収入が途切れるかもしれない不安が、一歩を踏み出すブレーキになっている。

2. **「一人で転職活動、何から始めれば…」**
   履歴書の書き方も面接対策もわからない。求人サイトを眺めるだけで時間が過ぎていく。誰かに相談したいけど、周りに転職経験者がいない。

3. **「入ってみたらブラックだった、は絶対イヤ」**
   求人票の「アットホームな職場」は信じられない。写真とテキストだけじゃ、本当の雰囲気はわからない。また同じ失敗を繰り返したくない。

4. **「今の仕事、このままでいいのかな…」**
   成長している実感がない。頑張っても評価されない。でも辞める勇気もない。気づけば同じ不満を抱えたまま、また1年が過ぎていく。

5. **「転職先でまた人間関係に悩んだら…」**
   新しい職場に馴染めるか不安。入社後に相談できる人がいない。結局また一人で抱え込むことになるんじゃないか。

**セクション末のつなぎ文**:
```
一つでも当てはまるなら、ツナゲルに相談してみてください。
```

#### レイアウト

**PC版**:
- 白背景（#FFFFFF）
- 見出し中央揃え
- 悩みは縦1列配置（左に赤丸のチェックアイコン、右にテキスト）
- 各悩みの間に薄いボーダー
- カード型UIは使わない。シンプルにリストとして並べる

**SP版**:
- 同じ縦1列レイアウト
- テキストサイズ微調整のみ

#### ビジュアル

- アイコン不使用（テキストの力で共感させる）
- 各悩みの見出し部分（「」で囲まれた部分）を太字 + テキスト主色
- 説明文はテキスト副色（#636E72）

#### デザイン詳細

- セクションパディング: PC 100px 0 / SP 64px 0
- 見出し: 32px/24px、#2D3436、中央揃え、下に40pxマージン
- 各悩み項目: 左にアクセントバー（3px幅、#FF6B35）、パディング 24px
